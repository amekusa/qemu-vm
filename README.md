# QEMU VM Starter
A simple shell script to setup a QEMU VM.

## Usage
```sh
# Clone this repo
git clone https://github.com/amekusa/qemu-vm.git MyVM

cd MyVM
./start.sh --help
```

```
QEMU VM Starter

Options:
  -h, --help    : Show this help
  -n, --dry-run : Dry-run mode
  -i, --install : Install mode
  -c, --conf    : Edit user.conf
  -r, --reset   : Reset user.conf

```
